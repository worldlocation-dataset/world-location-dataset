Quick Start for Developers

JavaScript (fetch countries)

fetch('https://worldlocation-dataset.github.io/world-location-dataset/data/countries.json')
  .then(res => res.json())
  .then(countries => {
    console.log('Total countries:', countries.length);
    console.log('First country:', countries[0]);
  });

JavaScript (fetch cities for one country)

const iso2 = 'IN'; // India
fetch(`https://worldlocation-dataset.github.io/world-location-dataset/data/cities/${iso2}.json`)
  .then(res => res.json())
  .then(cities => {
    console.log(`Cities in ${iso2}:`, cities.length);
    console.log('Example:', cities[0]);
  });

Python (requests)

import requests

url = "https://worldlocation-dataset.github.io/world-location-dataset/data/cities/US.json"
resp = requests.get(url)
cities = resp.json()
print("Cities in US:", len(cities))
